
## Usage

Register the plugin.

```typescript
import install from '@vuecs/gravatar';
import { createApp } from 'vue'

const app = createApp({})

app.use(install, {
  /* optional options */
})
```

After the component is registered, it can be used as follows.

```vue
<template>
    <VCGravatar
        :email="'max.mustermann@example.com'"
        :default-img="'retro'"
        :size="80"
        :rating="'g'"
    />
</template>
```
