## Installation

```
$ npm i --save @vue-layout/utils
```

## Usage

The following **form** helpers are provided:

- `buildFormInput`
- `buildFormInputCheckbox`
- `buildFormInputText`
- `buildFormSelect`
- `buildFormSubmit`
- `buildFormTextarea`

The following **list** helpers are provided:

- `buildList`
- `buildListActionRefresh`
- `buildListHeader`
- `buildListItem`
- `buildListItems`
- `buildListNoMore`
- `buildListPagination`
- `buildListSearch`
- `buildListTitle`
