```
$ npm i --save @vuecs/list-controls
```

## Usage

The following helpers are provided:

- `buildList`
- `buildListActionRefresh`
- `buildListFooter`
- `buildListHeader`
- `buildListItem`
- `buildListItems`
- `buildListNoMore`
- `buildListPagination`
- `buildListSearch`
- `buildListTitle`
